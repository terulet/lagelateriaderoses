PACK SEO MULTIDIOMA - LA GELATERIA DE ROSES

Archivos incluidos:
- index.html (ES)
- /ca/index.html
- /en/index.html
- /fr/index.html
- /nl/index.html
- /de/index.html
- /assets/img/ (imágenes extraídas del HTML para cargar más rápido)
- sitemap.xml con hreflang
- robots.txt
- CNAME

Cómo subirlo a GitHub:
1) Descomprime este ZIP.
2) Entra en github.com/lagelateriaderoses-prog/lagelateriaderoses.
3) Sube todos los archivos y carpetas del ZIP al repositorio.
4) Pulsa Commit changes.
5) Espera a que GitHub Pages actualice la web.
6) En Google Search Console, envía sitemap.xml y solicita indexación para:
   https://lagelateriaderoses.com/
   https://lagelateriaderoses.com/fr/
   https://lagelateriaderoses.com/nl/
   https://lagelateriaderoses.com/de/
   https://lagelateriaderoses.com/en/
   https://lagelateriaderoses.com/ca/

Notas SEO:
- Cada idioma tiene URL propia, canonical propio y hreflang.
- Las imágenes ya no están incrustadas como base64 dentro del HTML: están en assets/img.
- Esto mejora carga, mantenimiento y lectura de la web.
